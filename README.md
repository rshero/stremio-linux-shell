## âš™ï¸ Configuration

### MPV Config
Located at: `~/.local/share/stremio/mpv-portable/`

**mpv.conf** - Video settings:
```conf
vo=gpu-next                # Modern GPU renderer
hwdec=auto-safe           # Hardware decoding
profile=gpu-hq            # High quality preset
scale=ewa_lanczossharp    # Sharp upscaling
interpolation=yes         # Motion smoothing
deband=yes                # Reduce banding
```

**input.conf** - Custom keybindings (not needed, handled by native shortcuts)

### Discord Config
Located at: `~/.local/share/stremio/discord.json`

```json
{
  "enabled": true
}
```

Set to `false` to disable Discord Rich Presence.

## ğŸ› Troubleshooting

### Discord Not Showing
1. Make sure Discord is running
2. Check config: `cat ~/.local/share/stremio/discord.json`
3. Enable: Set `"enabled": true` and restart

### Shaders Not Working
1. Check if shaders are installed: `ls ~/.local/share/stremio/mpv-portable/shaders/`
2. Try clearing cache: `rm -rf ~/.local/share/stremio/cef/cache/*`
3. Reinstall: `./run-stremio.sh`

### Video Not Playing
1. Check MPV config: `cat ~/.local/share/stremio/mpv-portable/mpv.conf`
2. Try without hardware decoding: Set `hwdec=no` in mpv.conf
3. Check logs in terminal

### ThumbFast Not Working
1. Verify installation: `ls ~/.local/share/stremio/mpv-portable/scripts/`
2. Should see `thumbfast.lua`
3. Check config: `~/.local/share/stremio/mpv-portable/script-opts/thumbfast.conf`
